### Awsume (on windows)

1. Install it with pip
2. Run it only in **BASH-console** NOT any **CMDER**

###### use this one before go further
```sh
alias awsume='. awsume'
```

###### then command itself
```sh
awsume [profile]

# for refresh creds
awsume [profile-name] --refresh
```
